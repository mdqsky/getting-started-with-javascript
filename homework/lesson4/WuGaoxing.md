## W3school中学到的函数
函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。
函数就是包裹在花括号中的代码块，前面使用了关键词function：

function functionname()
{
这里是要执行的代码
}

在使用 return 语句时，函数会停止执行，并返回指定的值。

调用带参数的函数
在调用函数时，您可以向其传递值，这些值被称为参数。
这些参数可以在函数中使用。
